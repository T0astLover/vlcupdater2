# Prompter brukt (for transparens)

Under er konkrete prompt-formuleringer brukt for å få AI til å lage prosjektet:

1. "Lag et Python-program som sjekker siste VLC-versjon fra en nettside/API, sammenligner med installert versjon, og lagrer resultatet i SQLite."
2. "Lag en CLI med kommandoene `init-db`, `check` og `history`."
3. "Skriv tester for versjonssammenligning, parsing av versjon fra HTML, og databaseinnsetting."
4. "Skriv en README på norsk med kjøreinstruksjoner og eksempel-output."
5. "Gjør løsningen robust ved nettverksfeil, og logg feilstatus i databasen i stedet for å krasje."

Dette dokumentet kan legges ved intern dokumentasjon for å vise hvordan AI ble brukt i utviklingen.
